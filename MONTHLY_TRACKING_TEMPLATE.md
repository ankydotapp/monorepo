# Monthly Crypto Revenue Tracker — Anky, Inc.

This is a simple template for tracking onchain revenue. Fill it in monthly. No CPA required — just discipline.

## How to Use

1. Copy this file as `MONTHLY_2026-07.md` (or whatever month)
2. Fill in every transaction that touches ANKY_BANKR_FEES
3. Get USD values from Coingecko or Dexscreener at the time of the event
4. Keep it boring and accurate
